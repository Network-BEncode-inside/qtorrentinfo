## Description
Torrent file info. 

### Build
Clone the latest development source from GitHub.
Building qMakeTorrent requires a Qt4 development environment.

    $ git clone https://github.com/Network-BEncode-inside/qtorrentinfo
    $ cd qtorrentinfo
    $ qmake
    $ make
